## Overview

`LEFlatButton` is a `UIButton` subclass with a flat style

## Usage

### Installation

LEFlatButton is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "LEFlatButton"
```
